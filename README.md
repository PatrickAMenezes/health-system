# health-system
This project is an initiative to promove the open source to C++ community, so stay free to help if you want, especially if you're starting in the area, maybe it can be useful to you :)
------------------------------------------------------------------------------------------------
So contextualizing this system is like a "remake" of a project that I made for my degree, initially made with C, now it will be mainly made with C++.

The objective is create a system which can approach the different areas of a hospital, like the register of pacients, the administration of resources and the human resources control, for example.

Initially it will be just a desktop application, since the focus is create this project using C++, but we can expand it in the future for web and mobile too.

To better understand the different parts of this project, you can check some diagrams avaiable in the same name folder, and don't be afraid to change or add a business rule too if you have a better idea.
